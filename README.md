# 💰 Expense Tracker App

A responsive and user-friendly Expense Tracker web application built using React.  
It helps users track income and expenses efficiently with authentication, filtering and optimized performance using React Hooks.

---

## 🧰 Tech Stack

- Frontend: React.js, JavaScript, CSS
- State Management: useState, useEffect
- Performance Optimization: useMemo, useCallback
- API: MockAPI / JSON Server
- Authentication: Custom Login System
- Other Tools: Axios, React Hooks

---

## ✨ Features

- 🔐 User Login / Authentication system
- ➕ Add income and expenses
- 🗑️ Delete transactions
- 📊 Real-time balance calculation
- 🔎 Filter transactions (All / Income / Expenses)
- 📡 Fetch & display data from Mock API
- ⚡ Optimized rendering using useMemo & useCallback
- 📱 Fully responsive UI for all devices
- 🧠 Form focus handling using useRef
- 💾 Persistent data storage via API
